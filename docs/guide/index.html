简介

Toobit Agent Trade Kit 将 AI 助手与您的 Toobit 交易账户直接连接。无需在 AI 和交易所之间来回切换，只需用自然语言描述您的需求，AI 即可自动执行。

它以本地进程的方式运行在您的设备上。您的 API Key 永远不会离开本地。完全开源，遵循 MIT 协议。

Toobit 官方 AI 智能交易工具包。让您的 AI Agent 通过自然语言，在 Toobit 上交易现货、USDT-M 永续合约等多种产品。

📦 GitHub npm: toobit-trade-mcp npm: toobit-trade-cli

支持哪些功能？

1. 行情数据价格、订单簿、K线、资金费率、持仓量、标记价格、指数价格
2. 现货交易下单、撤单、批量下单、批量撤单、查询订单与成交
3. USDT-M 永续合约开多/开空、平仓、杠杆管理、保证金调整、止盈止损、闪电平仓、反向开仓
4. 账户管理余额查询、资金流水、子账户管理、划转、充提记录、API Key 检查
5. 审计日志本地工具调用记录，支持追踪与回溯

三种使用方式

方式说明
MCP 服务器
toobit-trade-mcp		 接入 Claude、Cursor、VS Code 或任意支持 MCP 的 AI 客户端，让您的 Agent 通过自然语言调用 Toobit 交易工具。

快速开始

第一步 — 安装

npm install -g @toobit_ai/toobit-trade-mcp @toobit_ai/toobit-trade-cli

验证安装：

toobit market ticker --symbol BTCUSDT
行情数据可立即使用，无需 API Key。

第二步 — 添加 Toobit API 凭证

创建配置文件 ~/.toobit/config.toml：

# ~/.toobit/config.toml
default_profile = "live"

[profiles.live]
api_key    = "your-api-key"
secret_key = "your-secret-key"

# 子账户配置（可选）
# [profiles.sub]
# api_key    = "sub-account-api-key"
# secret_key = "sub-account-secret-key"

获取 API Key：登录 toobit.com → 设置 → API 密钥 → 创建新 API Key

切勿将您的 API Key 或 Secret Key 粘贴到 AI 对话框中。所有凭证应仅保存在本地配置文件中。建议使用子账户的 API Key并仅开启所需的最小权限。

第三步 — 连接 AI 客户端

在项目根目录创建 .cursor/mcp.json（以 Cursor 为例）：

{
  "mcpServers": {
    "toobit-trade": {
      "command": "toobit-trade-mcp",
      "args": ["--modules", "all"]
    }
  }
}

支持的 AI 客户端：

客户端配置方式
Cursor.cursor/mcp.json
Claude Desktopclaude_desktop_config.json
VS Code (Copilot).vscode/mcp.json
Windsurf~/.codeium/windsurf/mcp_config.json
Claude Codeclaude-code mcp add

命令行

命令行工具是独立的终端工具，无需 AI 客户端。安装后即可通过 toobit 命令使用。

命令示例

# ── 行情数据 ──
toobit market ticker --symbol BTCUSDT
toobit market ticker-24hr --symbol ETHUSDT
toobit market candles --symbol BTCUSDT --interval 1h --limit 10
toobit market depth --symbol BTCUSDT --limit 5
toobit market funding-rate --symbol BTC-SWAP-USDT

# ── 现货交易 ──
toobit spot place --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001
toobit spot place --symbol DOGEUSDT --side BUY --type LIMIT --quantity 100 --price 0.08
toobit spot open-orders
toobit spot cancel --orderId 123456789

# ── 合约交易 ──
toobit futures place --symbol BTC-SWAP-USDT --side BUY_OPEN --type MARKET --quantity 1
toobit futures positions
toobit futures leverage --symbol BTC-SWAP-USDT

# ── 账户 ──
toobit account info
toobit account balance-flow

# ── 管道与脚本 ──
toobit market ticker --symbol BTCUSDT --json | jq '.data'
toobit market candles --symbol BTCUSDT --interval 1h --limit 200 --json | python3 analyze.py

安全

Toobit Agent Trade Kit 提供多层安全防护：

安全层说明
只读模式
--read-only		仅允许数据查询，所有写入工具被禁用，AI 无法执行任何交易操作。
模块过滤
--modules		精确控制暴露给 AI 的功能模块。例如 --modules market 仅暴露行情数据。
智能注册服务器启动时检测 API Key 权限。如果 Key 不具备交易权限，下单工具不会注册。
风险标签所有涉及资金操作的工具均标记为 [CAUTION]，提示 AI 在执行前确认。
本地限流内置令牌桶限流算法，防止对 Toobit API 造成过载。
审计日志所有工具调用自动记录到 ~/.toobit/logs/，敏感数据（签名等）自动脱敏。
本地运行所有程序在本地运行，签名在本地完成，凭证永远不会发送到第三方。
凭证安全：切勿将您的 API Key 或 Secret Key 透露给任何 AI 模型或粘贴到对话框中。所有凭证应仅保存在本地配置文件（~/.toobit/config.toml）中。强烈建议使用子账户的 API Key 并仅开启所需的最小权限。

常见问题

Agent Trade Kit 能做什么？

它覆盖了 Toobit 的核心交易功能：查询行情、交易现货和 USDT-M 永续合约、管理账户和资金。这一切都可以通过自然语言（MCP）或命令行完成。

支持哪些 AI 客户端？

支持所有兼容 MCP（Model Context Protocol）协议的 AI 客户端，包括 Claude Desktop、Claude Code、Cursor、VS Code 以及各类自定义 Agent。

这个工具是免费的吗？

是的。Agent Trade Kit 基于 MIT 协议完全开源且免费。你只需要一个 Toobit 账户和 API Key 即可进行交易（如果仅查询行情数据，甚至不需要 API Key）。

我的 API Key 安全吗？

所有程序都在本地运行。您的密钥仅保存在本地配置文件中，签名过程完全在本地完成，AI 永远无法获取您的敏感凭证。除 Toobit 官方服务器外，没有任何数据会被发送到第三方。代码完全开源，欢迎随时查看。

如果 AI 犯错，我会损失资金吗？

我们内置了多层安全机制。您可使用 --read-only 模式限制 AI 仅查询数据。AI 可能会犯错，请在执行前独立核对所有订单信息。用户需自行承担交易带来的损失。

现货交易对的 symbol 格式是什么？

现货使用 BTCUSDT、ETHUSDT 格式。合约使用 BTC-SWAP-USDT、ETH-SWAP-USDT 格式。

最小下单金额是多少？

取决于交易对。可以通过 market_get_exchange_info 工具查询具体的 MIN_NOTIONAL（最小名义金额）和 LOT_SIZE（最小/最大数量、步进）。大多数交易对的最小下单金额为 5 USDT。

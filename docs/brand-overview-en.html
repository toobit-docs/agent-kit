Product Components
Three components for different audiences
🔌

MCP Server

For AI app developers

A bridge that enables Claude, Cursor, VS Code, and other AI assistants to understand and execute Toobit trading commands.

⌨️

CLI

For quant & technical users

A command-line tool for direct terminal trading. Supports piping, cron jobs, and scripting for full automation.

🧩

Agent Skills

For AI Agent platforms

Plug-and-play skill cards. Once loaded, the AI agent instantly knows how to query markets, place orders, and manage accounts on Toobit.
